HTML Tables

Table tags

Tag Name Description
<table> Table Opens the table space.
<tdead> Table Head Defines the section of the table with the header row(s).
<tbody> Table Body Defines the section of the table with the content.
<tfoot> Table Footer Defines the bottom section of the table.
<tr> Table Row Defines one row of the table. Use it in the head, body and foot of the table.
<td> Table Head Entry Defines one entry in a row of the table head.
<td> Table Data Entry Defines one entry in the body of the table.
There are many more HTML tags, but these are the most important ones for tables.
